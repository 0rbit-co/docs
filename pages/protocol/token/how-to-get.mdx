# How to get $0RBT?

To test out 0rbit in your `ao` process, you can use the [`Sponsored Request`](/developer/get-request#sponsored-request) without needing to pay the __$0RBT__.

<Callout type="info" emoji="ℹ️">
    `Sponsored Request` method provides 50 requests per `ao` process per day.

    If you are interested in processing more than 50 requests, you will need to get the __$0RBT__ token. You can get the __$0RBT__ through the following:
</Callout>


## Complete the Quest

Completing Quest is the best way to get the $0RBT if you are a developer. 

All you need to do is:
    - Select the Quest you want to complete from [0rbit Quests](https://0rbit.co/quests)
    - Complete the selected Quest using the [`Sponsored Request`](/developer/get-request#sponsored-request) method.
    - Claim your $0RBT, from https://testnet-claim.0rbit.co at the end of the month.

## Swapping $AOCRED for $0RBT

If you have $AOCRED, you can swap them on [Bark](https://bark.arweave.dev) for $0RBT.
