# Introduction

0rbit is the __Decentralized Oracle Network__ built on `ao` to provide access to __LIMITLESS DATA__ on any `ao` process.

In the context of 0rbit, __LIMITLESS DATA__ means that an `ao` process can access _any data_ from _any valid source_.
